# 📝 Resume Analyzer AI - ASP.NET Core Web API & LM Studio

🚀 A powerful AI-powered Resume Analyzer built with **ASP.NET Core Web API, Entity Framework, and LM Studio (Deepseek-R1-Distill-Qwen-7B)**.  

This project extracts text from **PDF and DOCX** resumes, analyzes it using a **local AI model (LM Studio)**, and stores structured feedback in **SQL Server**.

---

## 📌 Features
✅ **Upload Resumes** (PDF & DOCX)  
✅ **Extracts Text** using PdfPig & OpenXML  
✅ **Sends Text to Local AI (LLM)** for analysis  
✅ **Stores results in SQL Server**  
✅ **API documentation** with Swagger  

---

## 🛠 Tech Stack
- **Backend:** ASP.NET Core Web API (.NET 8)  
- **Database:** SQL Server + Entity Framework  
- **AI Model:** LM Studio (Deepseek-R1-Distill-Qwen-7B)  
- **Text Extraction:** PdfPig + OpenXML  
- **API Documentation:** Swagger
